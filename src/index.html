Lounge do Açaí

This project consists of a self-service system developed to enhance the efficiency and user experience at the Lounge do Açaí, located at Universidade Católica de Pernambuco.

The system allows users to place orders through a mobile application, which are then sent to the kitchen via a bluetooth printer. This integration optimizes the ordering process and improves communication between customers and staff.

Password Game

This project is a simple game where the user has to guess a password. The password is a string of 4 characters, which can be letters or numbers. The user has as many attempts to guess the password as they want.

The game is developed using HTML, CSS, and JavaScript. It is a great way to practice your programming skills and have fun at the same time.
